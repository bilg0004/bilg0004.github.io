HTML Tags

Description of Five HTML Tags

Header tag (<header></header>):
Represents introductory content on a webpage
Paragraph tag (<p></p>):
Represents a paragraph
Main tag (<main></main>):
Represents primary content of a page
Citation tag (<cite></cite>):
Represents a citation
Unordered list tag (<ul></ul>):
Represents an unordered list
